<h2>QUESTION</h2>
Develop an ASP.NET Web Forms application for an Online College Admission
System. The application should allow students to register by entering their personal
and academic details using appropriate server controls. Validate all mandatory fields
using suitable Validation Controls and display meaningful error messages. After
successful registration, display a confirmation message with the entered details.
Additionally, use an AdRotator Control to display rotating banners for college
events, admissions, scholarships, or placement drives.
